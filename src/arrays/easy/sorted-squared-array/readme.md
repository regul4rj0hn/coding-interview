# Sorted Squared Array

## Description
Write a function that takes in a non-empty sorted array of integers and returns a new array of the same length with the squares of the original integers sorted in increasing order.

## Sample Input</h3>
```
array = [1, 2, 3, 5, 6, 8, 9]
```

## Sample Output
```
[1, 4, 9, 25, 36, 64, 81]
```
